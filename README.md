# تجربة Stripe على Vercel

## طريقة الرفع (بدون حتى Terminal، عن طريق الموقع مباشرة)

1. اعمل حساب على https://vercel.com (فيه ربط مباشر بـ GitHub)
2. ارفع هذا المجلد كـ repository جديد على GitHub (أو استخدم "Import" واسحب الملفات مباشرة لو Vercel يدعم ذلك بحسابك)
3. من Vercel: New Project → Import من الـ repo → Deploy (بدون أي إعدادات إضافية، Vercel بيتعرف على مجلد `api/` تلقائيًا كـ serverless functions)
4. بعد الرفع رح ياخذك رابط مثل:
   ```
   https://stripe-test-vercel.vercel.app
   ```
5. افتح الرابط، حط المفاتيح التجريبية، وجرب الدفعة.

## طريقة أسرع (لو عندك Node على جهازك أو Vercel CLI)

```
npm i -g vercel
cd stripe-vercel
vercel
```
بيسألك أسئلة بسيطة (اسم المشروع، إلخ) وبعدها بيديك رابط مباشرة.

## من وين المفاتيح؟

- https://dashboard.stripe.com (وضع Test mode مفعّل تلقائيًا، مجاني بالكامل)
- Publishable key: `pk_test_...`
- Secret key: `sk_test_...`
